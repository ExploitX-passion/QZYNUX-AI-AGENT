Qzynux GUI uses **PyQt6** which is dual-licensed:

## Qzynux Distribution Model

Qzynux is distributed as two separate components:

### 1. QZYNUX GUI (Open Source - GPL v3)
- Available for free on GitHub
- Uses PyQt6 for interface
- Anyone can view, modify, and redistribute the GUI code.

### 2. QZYNUX CORE (Proprietary - Commercial License)
- AI engine, security tools, and backend functionality
- Encrypted/obfuscated for protection
- Sold separately with commercial license
- Does NOT directly use PyQt6.

## Why This Model?

- The GUI is open source to comply with PyQt6 GPL requirements.
- The core remains proprietary to protect intellectual property.
- Users can use the open GUI with their own backend.
- Commercial users purchase the core for full functionality.
