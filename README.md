## Overview

Qzynux is a **privacy-first, dual-mode AI assistant** designed for security researchers, ethical hackers, and power users.

- Installed **locally** on your system (recommended 50GB space)
- Works offline AND online (dual-mode)
- Internet only required for license validation and updates
- Understands 170+ languages including Hindi & English naturally
- Controls terminals & security tools
- Never sends your data anywhere

---

## Feature Summary

**Core Intelligence**
Natural language understanding, context memory (24h), intent parsing, self-learning.

**Voice & Audio**
Wake word "Qzynux", offline STT (Whisper), TTS (pyttsx3), Hindi-English bilingual.

**Terminal Mastery**
Run any command, parse output, multi-terminal orchestration, man page analysis.

**Tool Integration**
Nmap (scripts, vuln scan), Metasploit (exploits, payloads), Wireshark (packet analysis), Burp Suite and other all system-wide tools.

**Security & Privacy**
3-tier permission, payment safety guard, port monitoring, audit logs, encrypted storage.

**Antivirus Engine**
Custom AV with YARA rules, real-time file monitoring, quarantine manager.

**Local Services**
SSH/MySQL/PostgreSQL diagnostics, Docker management, Git automation.

**Media & Creative**
Music control (Spotify), video/image editing (offline), screen recording.

**File Intelligence**
PDF/Word/Excel parsing, OCR, code analysis, document summarization.

**Situation Awareness**
Screen detection, cursor tracking, active window detection, clipboard monitoring.

**Search & Dorks**
Google dork generator, university paper guesser, background scraper.

**Math Engine**
Fast calculator, factorial, trig, complex expression solver.

**Multilingual**
100+ language detection (approximately 176), native expressions (ha ha ha / हा हा हा / jaja).

**Visualization**
Network maps, vulnerability heat maps, daily reports, timeline views.

**Plugin System**
Community plugins, sandboxed execution, permission-based install.

**Automation**
Cron jobs, scheduled tasks, conditional triggers, workflow automation.

**Note: The features listed above represent only a fraction of QZYNUX's capabilities. In total, QZYNUX ships with 375+ features covering every aspect of automation, security, and productivity.**
